## Interswitch Assessment

A simple app that receives data via several input fields and displays the data on a table. The application is hosted online and can be viewed here: https://the-switch.vercel.app/

### Tools used 
- React (Next.js)
- Chakra UI
- Redux
- MUI Datatable
- Sweetalert
- Vercel

### Functionalities
- Add users
- Edit users
- Delete users
